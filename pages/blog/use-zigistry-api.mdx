# Zigistry's API

You can now start using Zigistry's api as follows:

### For Packages:

**Get details for indivisual packages**

[https://zigistry.dev/api/packages/travisstaloch/simdjzon](https://zigistry.dev/api/packages/travisstaloch/simdjzon)

**Get Search results:**

[https://zigistry.dev/api/search?q=Zap](https://zigistry.dev/api/search?q=Zap)

**Get Filtered results:**

[https://zigistry.dev/api/search?q=Zap&filter=http](https://zigistry.dev/api/search?q=Zap&filter=http)

**Get all results from a single topic:**

[https://zigistry.dev/api/search?filter=http](https://zigistry.dev/api/search?filter=http)

### For Programs:

**Get details for indivisual programs**

[https://zigistry.dev/api/programs/travisstaloch/roc-cli-platform-zig](https://zigistry.dev/api/programs/travisstaloch/roc-cli-platform-zig)

**Get Search results:**

[https://zigistry.dev/api/searchPrograms?q=Zap](https://zigistry.dev/api/searchPrograms?q=Zap)

**Get Filtered results:**

[https://zigistry.dev/api/searchPrograms?q=Zap&filter=http](https://zigistry.dev/api/searchPrograms?q=Zap&filter=http)

**Get all results from a single topic:**

[https://zigistry.dev/api/searchPrograms?filter=http](https://zigistry.dev/api/searchPrograms?filter=http)

### For Special Index details (Packages):

**Get most used packages (Ascending order):**

[http://zigistry.dev/api/indexDetails?section=mostUsed&range=0..100](http://zigistry.dev/api/indexDetails?section=mostUsed&range=0..100)

**Get latest used packages (Ascending order):**

[http://zigistry.dev/api/indexDetails?section=latestRepos&range=0..100](http://zigistry.dev/api/indexDetails?section=latestRepos&range=0..100)

### For Special Index details (Programs):

**Get most used programs (Ascending order):**

[http://zigistry.dev/api/indexDetailsPrograms?section=mostUsed&range=0..100](http://zigistry.dev/api/indexDetailsPrograms?section=mostUsed&range=0..100)

**Get latest used programs (Ascending order):**

[http://zigistry.dev/api/indexDetailsPrograms?section=latestRepos&range=0..100](http://zigistry.dev/api/indexDetailsPrograms?section=latestRepos&range=0..100)

Obviously, a lot of things need to be added to the api and I am currently working on it. But for now, this can be used for fetching basic stuff.
